Welcome in WEB Mini Touring Machine: Give two input numbers, click start and see how Mini Touring Machine does its operations.
For each step you can see how the tape looks like, the number of actual state, head position on a tape and the actual step number.

In the last line you can see decimal result and time of each operation in ms (each step takes 1 ms)
